# Loopera Frontend

React + Vite frontend приложения Loopera.

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Предпросмотр

```bash
npm run preview
```

## Структура

- `components/` - React компоненты
- `pages/` - Страницы приложения
- `hooks/` - Custom hooks
- `utils/` - Утилиты
- `types/` - TypeScript типы
- `assets/` - Статические ресурсы
